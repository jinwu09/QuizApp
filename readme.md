# backend
to initalize backend this does is setting up the myssql database
```
cd backend
npm i
npm run init 
```

to run it 
```
cd backend
npm run dev
```
to run it more reliable 
```
cd backend
npm run buildpm2
```

# Frontend
to initialize setup in frontend
```
cd frontend
npm i
```

to run frontend simple 
```
cd frontend
npm run dev
```
